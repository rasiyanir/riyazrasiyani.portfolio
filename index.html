City of Hill Valley

Pet Adoption Web Application - City of Hill Valley

Deployed an online pet adoption Website working in a group using Scrum methodology. The website allows admin to login and post the pets available for adoption. Admin can approve or deny the user adoption request. Admin and user can also search any pets based on the location or on various pets characteristics with or without login. Admin get notification for the pet adoption request and user gets notified if the pet is approved or denied.

Tools: HTML, CSS, Bootstrap, Java, SQL Server database, JDBC.

E-commerce Website

Sport goods - E-commerce website

Angular and JavaScript based e-commerce website having three roles of super-admin, admin and buyer. Super-admin can login and manage the admins. Admin can login and search, post or delete a product on the website. And Customer or Buyer can browse the products and add them to cart and check out.

Tools: C#, Web API, HTML5, CSS3, Angular4, Bootstrap, jQuery, MSSQL.

Friendsbook

Similar experience as Facebook

The First phase: worked on personal console project using java. Second phase: Worked in a team to build the same project using JSF. The project lets the user have a similar experience as Facebook. The user can register, login, browse through the timeline. The user can send a friend request, post status, like or comment on the post, send and receive a message to a friend. Look at the notifications, see the friend’s list and check their profile.

Tools: Java, JSF, HTML5, CSS3, JDBC, Bootstrap, MySQL.
